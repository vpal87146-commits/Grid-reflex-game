# Grid-reflex-game
A fun and interactive reflex game built using **HTML, CSS, and JavaScript**.  
Test your reaction speed by clicking the active box before it disappears!

---

## 🚀 Features
- 3x3 grid layout
- Random box activation every 2 seconds
- 1-second reaction time to score
- Real-time score tracking
- 30-second game timer
- Instant visual feedback

---

## 🕹️ How to Play
1. Open the game in your browser
2. Watch for the box that turns green
3. Click it **before it disappears**
4. Earn 1 point for each correct click
5. Try to get the highest score in 30 seconds!

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation & timers)
